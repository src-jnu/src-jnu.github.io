Welcome to the SRC Lab.


Led by Taejune Park the SRC (Secure and Reliable Computing) lab at Chonnam National University focuses on network and system security. In particular, we study a variety of topics including cloud network (e.g., software-defined networking and network virtualization) security, embedded system and IoT. We are now working on Design and Implementation of high-performance network security systems and low-latency communications.
